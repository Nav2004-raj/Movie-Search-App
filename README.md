# 🎬 Movie Search App (React)

A simple and interactive **Movie Search App** built using **React**. This project helps you search for movies and view details like title, poster, and release year while learning core React concepts.

---

## 🚀 Features

* 🔍 Search movies in real-time
* 🎥 Display movie posters and details
* ⚡ Fast and responsive UI
* 📦 Built with reusable React components
* 🌐 Fetch data from an external movie API
* User can add movies to favourites page

---

## 🛠️ Tech Stack

* **React**
* **JavaScript (ES6+)**
* **CSS
* **OMDb API**

---
